# 🌤️ Værmelding App

En enkel og brukervennlig applikasjon som viser oppdatert værinformasjon for valgte byer.
Appen henter data fra en ekstern vær-API og presenterer temperatur, vind, fuktighet og værforhold i et moderne og responsivt grensesnitt.

# 🚀 Funksjoner

 - Søk etter værmelding for hvilken som helst by

 - Viser temperatur, værbeskrivelse, vindstyrke og luftfuktighet

 - Dynamiske værikoner basert på forhold (sol, regn, skyer osv.)

 - Automatisk oppdatering basert på brukerens posisjon 
 
 - Responsivt design for mobil, nettbrett og desktop

 - Ingen innlogging kreves
